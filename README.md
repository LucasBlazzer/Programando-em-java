# Programando-em-java
Fazendo meu primeiro programa java.
Agora que você aprendeu alguns conceitos importantes sobre entrada e saída de dados e atribuição de variáveis, você está pronto para fazer um programa Java para interagir com o usuário.

Neste exercício, você terá de realizar os seguintes passos no seu algoritmo usando o console da IDE para interagir com o seu programa:

-> Dê boas vindas a um usuário no console.
-> Logo depois, peça ao usuário alguns dados: nome completo, cpf, idade, peso, altura e estado civil.
-> Você terá de receber cada um dos dados e exibir -los de acordo com a descrição do execício.

Dicas: atente-se ao tipo de variável que você precisa receber e preste atenção nas diferenças de sintaxe em para receber cada tipo.
